## ðŸ“¦ Installing from ZIP

### Step 1: Extract the ZIP
1. Download `ShadowAI-v1.0.0.zip`
2. Extract all files to a folder (e.g., `ShadowAI`)
3. Remember the folder location

### Step 2: Load in Chrome
1. Open Chrome and go to `chrome://extensions`
2. Enable **"Developer mode"** (toggle in top right corner)
3. Click **"Load unpacked"**
4. Select the extracted `ShadowAI` folder
5. The extension will appear in your toolbar

### Step 3: Get Your API Key
1. Go to [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Sign in with your Google account
3. Click **"Create API Key"**
4. Copy the generated key

### Step 4: Configure ShadowAI
1. Click the ShadowAI icon in Chrome toolbar
2. The sidepanel will open
3. Click **"Ajustes"** (Settings)
4. Paste your API Key
5. Click **"Cerrar"** (Close)

### Step 5: Start Using!
- Type a message and press Enter
- Click the camera icon ðŸ“· to capture screenshots
- Paste images with Ctrl+V
- Create custom shortcuts in Settings
